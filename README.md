Back-end do projeto (C#)

1. Descrição

Este projeto é um estudo de caso na linguagem C# a partir do desenvolvimento de um sistema backend utilizando DotNET que fornece funcionalidade e simulação para criação e gerenciamento de cadastros de pessoas físicas e jurídicas.

2.FUNCIONALIDADES Este projeto possui funções para cadastrar pessoas físicas e jurídicas, simular cálculo de impostos, verificar CPF/CNPJ e criar/manipular arquivos .txt.

3. Tecnologias usadas dotNET, Vs Code, GIT, C#

4. Organização do Projeto Este projeto é proposto e ministrado pelo SENAI-SP no curso FULL Stack DEVELOPMENT

5. Pré-requisitos de instalação Os pré-requisitos para instalação do projeto são as ferramentas dotNET e VScode para processamento e execução de arquivos.

6. A execução do aplicativo é realizada no console local apenas para fins de desenvolvimento.
